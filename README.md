# Telegram Video Downloader Bot

بوت تيليجرام يقوم بتحميل فيديوهات من YouTube, TikTok, Instagram باستخدام `yt-dlp` ويُرسلها للمستخدم.

## خطوات سريعة للرفع على GitHub + Render

1. أنشئ مستودع جديد على GitHub.
2. أنشئ الملفات المذكورة (`main.py`, `requirements.txt`, `Procfile`, `README.md`) وانسخ المحتوى.
3. اذهب إلى Render.com وانشئ مشروع جديد (Web Service) مرتبط بالمستودع.
4. اضف متغيرات البيئة في إعدادات الخدمة على Render:
   - `TELEGRAM_BOT_TOKEN` = توكن البوت من BotFather
   - `CHANNEL_USERNAME` = `@english_quotes_ar` (أو اسم قناتك)
5. **مهم جداً:** أضف البوت كـ مشرف (Admin) في القناة `@english_quotes_ar` حتى يتمكن من التحقق من الاشتراكات.
6. انشر الخدمة—Render سيقوم بتثبيت الاعتمادات ثم تشغيل البوت.

## ملاحظات وفِنى
- البوت يستخدم `yt-dlp` لدعم تنزيل من منصات متعددة.
- هناك قيود لحجم الملفات عند إرسالها عبر بوت تيليجرام (حوالي 50MB). الملفات الأكبر لن تُرسل عبر البوت مباشرة.
- تأكد من احترام حقوق النشر عند تنزيل الفيديوهات.

## التخصيص
- لتغيير قناة الاشتراك، غيّر متغير البيئة `CHANNEL_USERNAME` أو حرّر `main.py` مباشرة.
